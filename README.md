# fastapi-business-api-portfolio

## 概要
業務系サービスを想定した FastAPI による Web API のポートフォリオです。
認証、法人向け利用モード管理、利用オプション管理など、
実務で扱われる業務ロジックを含む API を段階的に実装していく予定です。

## 使用技術
- Python
- FastAPI
- MySQL
- Docker / docker-compose
- Poetry

## 実装予定の機能
- ユーザ認証・管理 API
- 法人向け利用モード管理 API
- サービス利用オプション管理 API
- 業務データ取得・履歴管理 API

※ 本リポジトリは実務経験を元に内容を抽象化したポートフォリオです。
