# Corporate Mobility Management API（Portfolio）

## 概要
本プロジェクトは、法人および個人ユーザ向けに  
モビリティサービスの利用管理を行う  
業務用 Web API のサンプル実装です。

実務で経験した業務 API 開発の構成・設計をもとに、  
ポートフォリオ用として再構成したデモプロジェクトです。  
※ 実在する企業・サービスとは一切関係ありません。

---

## 技術スタック
- Python
- FastAPI
- MySQL（想定）
- Docker
- Poetry（依存関係管理）

---

## 主な機能
### 認証・ユーザ管理
- ユーザパスワード認証 API
- サービス利用規約同意 API

### 法人利用管理
- 法人利用モード取得 API
- 法人利用モード承認 / 取消 API
- 法人コード登録 API

### 利用・車両管理
- 車両利用情報取得 API
- 利用モード切替 API

### オプション管理
- 時間貸しオプション登録 API
- 月額利用オプション継続 / 解約 API

### 共通処理
- アタッチメント受信履歴登録 API
- 共通フロー（Command パターン想定）

---

## アーキテクチャ
本プロジェクトでは、実務で経験した  
Clean Architecture / DDD の考え方を参考にし、

- Presentation
- Application（UseCase）
- Domain
- Infrastructure

の責務を分離した構成を採用しています。

なお、本リポジトリはポートフォリオ目的の
サンプル実装であり、
実務で使用したコードや
内部ライブラリは一切含まれていません。
